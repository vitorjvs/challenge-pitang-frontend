# challenge-pitang-frontend
# Installation
### Using yarn

1. Open this folder in bash

2. Run the command `yarn`

3. And finally run `yarn start` to run the project

### Using npm

1. Open this folder in bash

1. Run the command `npm i`

1. And finally run `npm run start` to run the project

# Test

### Using yarn

### Using npm
